# WhatsApp Auto Reply Bot 

## Overview

This project is a Python-based WhatsApp auto-reply bot.
It reads chat messages from the screen, processes the latest message, and automatically sends a reply using an AI model.



## Features

* Reads chat messages using screen automation
* Detects latest message
* Avoids duplicate replies
* Generates AI-based responses
* Automatically sends replies



## Technologies Used

* Python
* pyautogui (for screen automation)
* pyperclip (for clipboard handling)
* OpenRouter API (AI responses)


## How It Works

1. Selects chat messages from screen
2. Copies chat using keyboard automation
3. Extracts latest message
4. Sends message to AI model
5. Gets response
6. Pastes and sends reply automatically






## Limitations

* Works only on local machine
* Depends on screen coordinates
